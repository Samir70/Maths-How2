Pythagoras -- using the formula

Pythagoras' formula is used to find the third side of a right-angled triangle when two of the sides are already given

Pythagoras' formula is
a2 + b2 = c2
Where a and b are the short sides, and c is the long side.

There are three steps to using the formula:

  • Square the two sides you are given in the question
  • Either add or subtract the numbers you got in stage 1
  • Square root the number you got in the previous step

The only tricky part is the second. When do you add, when do you subtract?
Answer: Add if you are looking for the long side.

NB: you need to sides to start using Pythagoras.
If you have one side and an angle, look at trigonometry
